## Calles / pista

Placeholders PNG generados para que puedas reemplazarlos:

- `assets/calles/asfalto-recta.png`
- `assets/calles/meta-ajedrezada.png`
- `assets/calles/pista-base.png`

`pista-base.png` se usa como imagen principal de la pista y se estira al ancho disponible. La meta dinamica, autos y marcas de distancia se dibujan arriba con HTML/CSS para que sigan respondiendo a la distancia maxima.

Recomendacion: conservar esos nombres si queres que la app los tome sin tocar codigo.
